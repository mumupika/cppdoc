The interface of C standard library is defined by the following collection of headers.

<dl>
  <dt><CHeader name="assert" /></dt>
  <dd>Conditionally compiled macro that compares its argument to zero</dd>

  <dt><Revision noborder since="C99"><CHeader name="complex" /></Revision></dt>
  <dd>Complex number arithmetic</dd>

  <dt><CHeader name="ctype" /></dt>
  <dd>Functions to determine the type contained in character data</dd>

  <dt><CHeader name="errno" /></dt>
  <dd>Macros reporting error conditions</dd>

  <dt><Revision noborder since="C99"><CHeader name="fenv" /></Revision></dt>
  <dd>Floating-point environment</dd>

  <dt><CHeader name="float" /></dt>
  <dd>Limits of floating-point types</dd>

  <dt><Revision noborder since="C99"><CHeader name="inttypes" /></Revision></dt>
  <dd>Format conversion of integer types</dd>

  <dt><Revision noborder since="C95"><CHeader name="iso646" /></Revision></dt>
  <dd>Alternative operator spellings</dd>

  <dt><CHeader name="limits" /></dt>
  <dd>Ranges of integer types</dd>

  <dt><CHeader name="locale" /></dt>
  <dd>Localization utilities</dd>

  <dt><CHeader name="math" /></dt>
  <dd>Common mathematics functions</dd>

  <dt><CHeader name="setjmp" /></dt>
  <dd>Nonlocal jumps</dd>

  <dt><CHeader name="signal" /></dt>
  <dd>Signal handling</dd>

  <dt>
    <Revision
      noborder
      since="C11"
      traits={[ { trait: "deprecated", since: "C23" } ]}
    >
      <CHeader name="signal" />
    </Revision>
  </dt>
  <dd>alignas and alignof convenience macros</dd>

  <dt><CHeader name="stdarg" /></dt>
  <dd>Variable arguments</dd>

  <dt><Revision noborder since="C11"><CHeader name="stdatomic" /></Revision></dt>
  <dd>Atomic operations</dd>

  <dt><Revision noborder since="C23"><CHeader name="stdbit" /></Revision></dt>
  <dd>Macros to work with the byte and bit representations of types</dd>

  <dt>
    <Revision
      noborder
      since="C99"
      traits={[ { trait: "deprecated", since: "C23" } ]}
    >
      <CHeader name="stdbool" />
    </Revision>
  </dt>
  <dd>Macros for boolean type</dd>

  <dt><Revision noborder since="C23"><CHeader name="stdckdint" /></Revision></dt>
  <dd>Macros for performing checked integer arithmetic</dd>

  <dt><CHeader name="stddef" /></dt>
  <dd>Common macro definitions</dd>

  <dt><Revision noborder since="C99"><CHeader name="stdint" /></Revision></dt>
  <dd>Fixed-width integer types</dd>

  <dt><CHeader name="stdio" /></dt>
  <dd>Input/output</dd>

  <dt><CHeader name="stdlib" /></dt>
  <dd>General utilities: <Missing>memory management</Missing>, <Missing>program utilities</Missing>, <Missing>string conversions</Missing>, <Missing>random numbers</Missing>, <Missing>algorithms</Missing></dd>

  <dt><Revision noborder since="C29"><CHeader name="stdmchar" /></Revision></dt>
  <dd>Text transcode</dd>

  <dt>
    <Revision
      noborder
      since="C11"
      traits={[ { trait: "deprecated", since: "C23" } ]}
    >
      <CHeader name="stdnoreturn" />
    </Revision>
  </dt>
  <dd><Missing>noreturn</Missing> convenience macro</dd>

  <dt><CHeader name="string" /></dt>
  <dd>String handling</dd>

  <dt><Revision noborder since="C99"><CHeader name="tgmath" /></Revision></dt>
  <dd>Type-generic math (macros wrapping <CHeader name="math" /> and <CHeader name="complex" />)</dd>

  <dt><Revision noborder since="C11"><CHeader name="threads" /></Revision></dt>
  <dd>Thread library</dd>

  <dt><CHeader name="time" /></dt>
  <dd>Time/date utilities</dd>

  <dt><Revision noborder since="C11"><CHeader name="uchar" /></Revision></dt>
  <dd>UTF-16 and UTF-32 character utilities</dd>

  <dt><Revision noborder since="C95"><CHeader name="wchar" /></Revision></dt>
  <dd>Extended multibyte and wide character utilities</dd>

  <dt><Revision noborder since="C95"><CHeader name="wctype" /></Revision></dt>
  <dd>Functions to determine the type contained in wide character data</dd>
</dl>
